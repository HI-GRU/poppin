## 프로젝트 소개

* 프로젝트명: 개인 맞춤형 팝업 스토어 추천 플랫폼 **'팝핀(Popping)'**
* 서비스 특징: 하나의 문화로 자리잡은 팝업 스토어에 대한 정보들을 보다 쉽고 빠르게 제공한다.
* 주요 기능
  - 다양한 조건에 따른 팝업 스토어 조회 (랭킹순, 오픈 예정순 등)
  - 사용자 맞춤 팝업 스토어 추천
  - 팝업 스토어별 그룹 채팅
  - 팝업 스토어 예약 및 예약자 관리
* 주요 기술
  - WebSocket
  - JWT Authentication
  - REST API
  - Redis
* 배포 환경
  - URL: https://i11a806.p.ssafy.io/
  - 테스트 계정: sinphoto@naver.com / Rhakwnr2! (카카오 로그인)
